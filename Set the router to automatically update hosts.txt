刷老毛子华硕固件路由器设置每天自动更新hosts方法：
1.进入路由器管理界面
2.点击左侧菜单栏的“高级设置”→“系统管理”
3.“系统管理”界面选择“服务”栏
4拉倒最下边，打开“Cron守护程序 (计划任务)”开关。点击蓝字“计划任务 (Crontab)”进入
5.另起一行，添加以下代码进去即可。#号是注释内容不要复制进去。看自己需求选择源，不要两个都订阅，一般仅需订阅其中一个源即可。

#订阅ADFQ hosts：
0 0 * * * cd /etc/storage/dnsmasq;wget --no-check-certificate https://raw.githubusercontent.com/sy618/hosts/master/ADFQ;sudo rm -rf hosts;mv ADFQ hosts

#订阅纯AD hosts：
0 0 * * * cd /etc/storage/dnsmasq;sudo rm hosts;wget --no-check-certificate https://raw.githubusercontent.com/vokins/yhosts/master/hosts


（其他路由器的设置方法欢迎大家分享与提供补充完善！！）
